# Academia-Jedi-HW-17 Collection

[![Icestudio](https://img.shields.io/badge/collection-icestudio-blue.svg)](https://github.com/FPGAwars/icestudio)
![Version](https://img.shields.io/badge/version-v0.1.0-orange.svg)

Colección para la Academia Jedi de Hardware.

## Install

* Download the collection: [Academia-Jedi-HW-17.zip](https://github.com/Obijuan/digital-electronics-with-open-FPGAs-tutorial/raw/master/wiki/Tutorial-17/Collection/Academia-Jedi-HW-17.zip)
* Install the collection: *Tools > Collections > Add*
* Load the collection: *Select > Collection*
